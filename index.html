RationalRewards

Reasoning Rewards Scale Visual Generation at both Training and Test Time

From scalar scoring to critique-then-score optimization in both parameter space and prompt space.

Project Page Model (T2I) Model (Edit) SFT Data Code

Abstract

Most reward models for visual generation reduce rich human judgments to a single unexplained score, discarding the reasoning that underlies preference. RationalRewards produces explicit, multi-dimensional critiques before scoring, enabling two complementary optimization pathways: train-time RL with interpretable, fine-grained rewards and test-time Generate-Critique-Refine prompt updates without parameter changes.

To avoid costly rationale annotation, we introduce Preference-Anchored Rationalization (PARROT), a framework that recovers high-quality rationales from preference data via anchored generation, consistency filtering, and distillation.
